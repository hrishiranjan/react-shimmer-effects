# shimmer-react

> Complete shimmer effects for React


## Install

```bash
npm install --save shimmer-react
```

## Usage

```jsx
import React, { Component } from "react";

import ExampleComponent from "shimmer-react";

class Example extends Component {
  render() {
    return <ExampleComponent />;
  }
}
```
